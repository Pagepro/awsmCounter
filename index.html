'Awesome Counter' is a jQuery plugin.

It works on all modern browsers and it's very easy to use.

How to

Get the basic .css and .js add them to your project from here.

<link rel="stylesheet" href="awsm-counter.css">
<script src="awsm-counter.js"></>

Call the plugin on elements you want

$('.js-awsm-counter').awsmCounter();

Available options

  • duration: [number] time of animation (default=2000)
  • animation: [boolean] if true it will be animated (default=true)
  • ascend: [boolean] starting from 0 or 9 (default=true)
  • timeout: [number] the timeout time (default=0)
  • offsetTrigger: [number] how many pixels before the element it should start animation (default=1000)
  • seperator: [string] the seperator, if set to false there will be no seperator (default=',')
